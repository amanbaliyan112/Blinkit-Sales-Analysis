## Blinkit Sales Analysis Dashboard

An interactive **Power BI dashboard** that transforms Blinkit's sales data into actionable business insights through KPI tracking, sales analysis, customer behavior, inventory distribution, and outlet performance using **Power BI, SQL Server, Excel, DAX, and Power Query**.

---



# Project Overview

This project focuses on analyzing Blinkit's grocery sales data to understand business performance, customer purchasing behavior, and outlet efficiency. The dashboard provides decision-makers with meaningful insights through interactive visualizations and key performance indicators (KPIs).

The project demonstrates the complete data analytics workflow, including:

- Data Cleaning
- Data Transformation
- SQL Analysis
- Data Modeling
- DAX Calculations
- Interactive Dashboard Development
- Business Insights Generation

---

# Business Objective

The primary objective of this project is to analyze Blinkit's sales performance, customer satisfaction, and inventory distribution to identify trends and opportunities for business growth.

The dashboard enables users to:

- Monitor overall sales performance
- Compare outlet performance
- Analyze product categories
- Evaluate customer ratings
- Study outlet size and location performance
- Identify sales trends over time

# Data Cleaning

The dataset contained inconsistent values in the **Item Fat Content** column.

Examples:

- LF
- low fat
- Low Fat
- reg
- Regular

SQL queries were used to standardize these values for accurate reporting and visualization.

# Business Insights

The dashboard highlights several important business insights:

- Medium-sized outlets generated the highest sales.
- Tier 3 outlets contributed the largest share of revenue.
- Regular fat products generated more sales than low-fat products.
- Supermarket Type 1 achieved the highest overall sales.
- Fruits & Vegetables and Snack Foods were among the top-performing product categories.
- Average customer rating remained consistent across outlet types.
